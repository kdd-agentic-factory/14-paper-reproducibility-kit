# AGENTS.md — Paper Reproducibility Kit

## Repository Mission

This repository packages the academic paper, experiments, datasets, artifacts and
reproducibility evidence for the KDD-governed agentic race engineering ecosystem.

## Mandatory Rules

- Do not include sensitive telemetry.
- Do not include real credentials or API keys.
- Do not include private operational data.
- Do not create tables without source metadata.
- Do not create figures without generation scripts.
- Do not modify experimental results manually without recording provenance.
- Do not cite unavailable artifacts.
- Do not export reviewer packages with confidential information.
- All datasets must have a dataset card.
- All artifacts must appear in `artifacts/artifact-index.yaml`.
- All experiments must appear in `experiments/experiment-index.yaml`.

## Allowed Responsibilities

- compile the paper,
- store synthetic datasets,
- store processed experimental results,
- generate tables from data,
- generate figures from data,
- validate reproducibility metadata,
- package reviewer artifacts,
- maintain bibliography,
- document experiment-to-paper mapping,
- compute and verify checksums.

## Forbidden Responsibilities

- running production services,
- approving race engineering decisions,
- modifying security policies in other repositories,
- training production models,
- storing raw restricted telemetry,
- publishing confidential data.

## File Conventions

- Results: `YYYY-MM-DD_experiment-name_metric.csv`
- Artifacts: referenced by `artifact_id` from `artifacts/artifact-index.yaml`
- Datasets: must include a `dataset-card.md` and `generation-config.yaml`
- Figures: generated by `scripts/generate-figures.py`
- Tables: generated by `scripts/generate-tables.py`

## Before Closing a Contribution

- Verify that `paper/main.tex` compiles.
- Verify that all cited paths in the manuscript exist.
- Update `artifacts/artifact-index.yaml` for any new artifact.
- Update `experiments/experiment-index.yaml` for any new experiment.
- Run `make test` and confirm all tests pass.
- Compute checksums with `make validate`.
