# experiments/

Experiment results, configurations, and evidence imported from
`08-experimentation-lab`.

Each experiment directory contains:
- `config.yaml` — experiment configuration and parameters,
- `results.csv` — raw results per run,
- `evidence.json` — supporting evidence and metadata,
- `table.tex` — generated LaTeX table for the paper,
- `summary.md` — human-readable summary.

All experiments are indexed in `experiment-index.yaml`.
