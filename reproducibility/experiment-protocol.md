## Procedure

1. Prepare the environment.
2. Verify dependencies and versions.
3. Configure experiment parameters.
4. Run the experiment.
5. Export raw outputs to `results/`.
6. Compute metrics using the definitions in `metrics-definition.md`.
7. Generate paper figures and tables.
8. Record anomalies, failures, and deviations.
