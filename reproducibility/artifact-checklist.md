# Artifact Checklist

Use this checklist before submitting or releasing the paper artifact.

## Manuscript

- [ ] `paper/main.tex` compiles.
- [ ] All section files are included from `paper/main.tex`.
- [ ] Bibliography entries are complete.
- [ ] Figures referenced in the paper exist.
- [ ] Tables referenced in the paper exist.

## Reproducibility

- [ ] Docker reproduction instructions are complete.
- [ ] Kubernetes reproduction instructions are complete, if applicable.
- [ ] Experiment protocol includes commands and environment details.
- [ ] Metrics are defined before being reported.
- [ ] Random seeds and configuration files are documented.

## Results

- [ ] Raw results are stored or linked.
- [ ] Processed results are stored or linked.
- [ ] Generated figures and tables are traceable to source data.
- [ ] Deviations from the protocol are documented.

## Supplementary Material

- [ ] Appendices are complete.
- [ ] Notebooks execute from top to bottom.
- [ ] External dependencies and datasets are documented.
- [ ] License and citation information are included where needed.
