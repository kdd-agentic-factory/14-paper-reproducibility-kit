\section{Repository Organisation}
\label{app:repository-organization}

The full ecosystem comprises fifteen repositories under the
\texttt{kdd-agentic-factory} organisation. Each repository has a single declared
responsibility, a dedicated \texttt{AGENTS.md} governance file, and a defined
interface with adjacent repositories.
