\section{Introduction}

Agent-oriented software development has moved from isolated assistant usage to
coordinated workflows where multiple agents can plan, implement, review, and
validate software artifacts. This paper studies that transition through an
agentic factory architecture designed to connect development activity with a
reproducible scientific evaluation.

The goal of this repository is to make the paper auditable: each claim should be
traceable to a protocol, metric definition, experiment output, figure, table, or
appendix.

\subsection{Contributions}

The expected contributions are:

\begin{itemize}
  \item a reference architecture for an agentic software factory;
  \item an implementation model aligned with reproducible experiments;
  \item an experimental protocol for evaluating agentic workflows;
  \item a public artifact structure connecting code, results, and manuscript.
\end{itemize}
