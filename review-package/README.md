# Review Package

This directory contains the materials prepared for peer review and artifact
evaluation.

## Contents

| File | Description |
|------|-------------|
| `reviewer-guide.md` | Step-by-step guide for reviewers |
| `artifact-evaluation-checklist.md` | ACM/IEEE artifact evaluation checklist |
| `minimal-reproduction.md` | Fastest path to reproduce key results |
| `full-reproduction.md` | Complete reproduction instructions |
| `anonymized-artifacts/` | Anonymised subset of artefacts |

## For Reviewers

Start with `reviewer-guide.md`. For a quick sanity check of the main results,
follow `minimal-reproduction.md`.
