# datasets/

Synthetic and anonymised datasets used in the experiments.

All datasets are:
- publicly shareable (no confidential telemetry),
- documented with a dataset card,
- generated by scripts in `scripts/`,
- indexed in `dataset-index.yaml`.

## Dataset Classification

All datasets in this repository are classified as **public-synthetic**.
No real rider, team, bike, or circuit telemetry is included.

## Subdirectories

| Directory | Dataset ID | Description |
|-----------|-----------|-------------|
| `synthetic-telemetry/` | synthetic-telemetry-v1 | Simulated motorcycle telemetry |
| `workflow-runs/` | workflow-runs-v1 | Synthetic workflow execution logs |
| `rag-benchmarks/` | rag-benchmark-v1 | Retrieval evaluation queries |
| `copilot-prompts/` | copilot-prompts-v1 | Copilot prompt-response pairs |
| `simulation-scenarios/` | what-if-scenarios-v1 | Setup what-if scenarios |
